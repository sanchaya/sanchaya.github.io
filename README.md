# Sanchaya.page

A directory of [Sanchaya](https://sanchaya.org)'s Kannada language computing projects and resources.

Live at: [https://sanchaya.page](https://sanchaya.page)

## Adding a project

Edit [data.yml](data.yml) with the project details. The site auto-deploys on merge to `main`.

## Building locally

1. Clone the repo
2. `pip install -r requirements.txt`
3. `python build.py`

The static site is generated into the `site/` directory.
